# Projeto Daily Diet Fullstack

Projeto desenvolvido para a disciplina de Programação Web II.

Sistema de controle de dieta com Backend (Node.js/SQLite) e Frontend (React).

Fernando dos Reis Galvão | Polo de Arapoema-TO

## Como rodar o projeto

Clone o repositório e siga os passos abaixo:

### 1. Configurar o Backend

Entre na pasta backend, instale as dependências e inicialize o banco de dados:

```bash
cd backend
npm install

# Este comando cria o arquivo db.sqlite e as tabelas automaticamente:
npm run db:init

# Rodar o servidor (porta 3333):
npm run dev
```

### 2. Configurar o Frontend

Abra um novo terminal, entre na pasta frontend e rode o projeto:

```bash
cd frontend
npm install
npm run dev

# O projeto estará rodando em http://localhost:5173.
```
